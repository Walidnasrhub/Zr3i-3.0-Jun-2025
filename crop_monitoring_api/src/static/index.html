Crop Monitoring API

Welcome to the Crop Monitoring and Management API. This API provides endpoints for managing user accounts, field boundaries, and accessing satellite imagery data for agricultural monitoring.

API Documentation

Authentication

POST /api/auth/register

Register a new user account.

{
  "username": "user123",
  "email": "user@example.com",
  "password": "securepassword"
}

POST /api/auth/login

Login to an existing account.

{
  "username": "user123",
  "password": "securepassword"
}

Field Management

GET /api/fields

Get all fields for the authenticated user.

POST /api/fields

Create a new field with GeoJSON boundary.

{
  "name": "North Field",
  "description": "Corn field north of the farm",
  "crop_type": "corn",
  "boundary": {
    "type": "Feature",
    "geometry": {
      "type": "Polygon",
      "coordinates": [
        [
          [13.37, 52.52],
          [13.38, 52.52],
          [13.38, 52.53],
          [13.37, 52.53],
          [13.37, 52.52]
        ]
      ]
    }
  }
}

Satellite Data

GET /api/satellite/fields/{field_id}/satellite-images

Get all satellite images for a specific field.

POST /api/satellite/fields/{field_id}/fetch-satellite-data

Fetch new satellite data for a field.

{
  "start_date": "2023-06-01",
  "end_date": "2023-06-30",
  "cloud_cover": 10
}

GET /api/satellite/fields/{field_id}/vegetation-indices

Get vegetation indices (NDVI or EVI) for a field.

Query parameters:

  • start_date: Filter by start date (YYYY-MM-DD)
  • end_date: Filter by end date (YYYY-MM-DD)
  • index_type: Type of index (ndvi or evi, default: ndvi)
